What's new in Keepers of the Stones II version 1.0.3?<br />
<br />- Updated visual effects for the lightning element
<br />- Slightly enhanced metal abilities to compensate for slow weapons
<br />- The efficiency of the blood master's abilities has been improved
<br />- The Call of the Past record now gives the power lock effect while in the hands
<br />- Added arguments for the pw command - reset, master_effect_duration and recharge_time
<br />- Some add-ons designed for earlier versions of the mod (1.0.0-1.0.2) may not work on this version
<br />- [API] Colorful Powers API has been updated to version 20
<br />- [Bugfix] Fixed a bug where when the stone next to the keeper's box was activated, the game would freeze
<br />- [Bugfix] Fixed a bug where items in vaults could rotate quickly when placing the same blocks next to each other
<br />- [Bugfix] Fixed bugs related to ocean stone
